## 网易七鱼访客端 H5 接入示例

**一、本地启动**

```
npm install 或 yarn
```

```
npm start
```

浏览器打开`http://localhost:8217/index.html`


**二、说明**

- 请将 appid 、staffid、groupid 等设置为您企业的对应数据，以保证可以正常进线
  
- 本示例中包括「上报用户信息」、「商品卡片」等基础功能，更多功能 [详见](http://qiyukf.com/docs/guide/web/)
